使用须知 (v4)

这是一个 B 站 API 工具集，由 OpenClaw AI 大脑调用：

  • OpenClaw 自己调 LLM 决策，由本 skill 执行（点赞/投币/评论/弹幕/收藏/关注/私信）
  • 本 skill 不内置 AI — 只暴露 atomic 工具 + 阈值门控
  • 所有操作走 B 站限流（-799 自动退避）+ 全局冷却
  • Web 仅本机（127.0.0.1），密码 bcrypt 存 Data/web_auth.json
  • v4 新增：精力值系统（默认 20 / 每个视频 -1 / 耗尽后停 30 分钟）
